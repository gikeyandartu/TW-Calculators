Equipment Calculator

This calculator will tell you how many more grey materials you need, how long it will take to craft those, and when crafting would finish.
This does not take Sandtable or any special events that give materials into account.

Select the desired rarity of the equipment you are trying to build. Input the number of each color material you have.
If you are crafting gold, check the checkbox if it is applicable.
The Reset button will clear the page for you if you want to start over.
